Plugin to integrate the Rufus Chatbot. 
 
== Description ==
 
This plugin was created to facilitate the integration of the Rufus Chat on Wordpress sites. For it to work,
you must already have a Rufus account.
 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Install the plugin
2. Go to /wp-admin/options-general.php?page=rufus-wordpress and enter your Rufus Key
